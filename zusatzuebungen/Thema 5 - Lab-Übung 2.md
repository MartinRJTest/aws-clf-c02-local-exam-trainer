# Thema 5 - Lab-Übung 2

Tag: Tag 3

Thema: Konzepte der Cloud-Wirtschaftlichkeit

Übung: S3 Lifecycle als Kostenhebel

Verwendetes Go-Deploy-Lab: Lab 11

Erstelle in Lab 11 einen S3 Bucket, lade ein Objekt hoch und lege eine Lifecycle Rule an. Achte besonders auf die Lifecycle rule actions: transition actions verschieben Objekte später in eine andere Storage Class, expiration actions löschen Objekte nach einer festgelegten Zeit. Notiere die gewählte Regel und ob sie enabled ist.

Übertrage das Prinzip auf drei Dateitypen der Logistik Huber GmbH: aktueller Lieferschein, alter Scan und gesetzlich aufzubewahrendes Archiv. Ordne jedem Beispiel eine passende Speicherstrategie zu, etwa S3 Standard, S3 Standard-IA oder Glacier. Begründe in je einem Satz, warum Zugriffshäufigkeit und Aufbewahrungsdauer die Kosten beeinflussen. Falls die Lab-Konsole eine Option nicht anbietet, beschreibe die gewünschte Regel schriftlich.
