# Thema 4 - Lab-Übung 2

Tag: Tag 3

Thema: Migration zur AWS Cloud

Übung: Datenbank-Zielsystem für Migration

Verwendetes Go-Deploy-Lab: Lab 08

Erstelle in Lab 8 die RDS MySQL Database. Betrachte die Datenbank dabei als mögliches Ziel für eine Migration: Eine bestehende MySQL-Datenbank soll nicht mehr auf einem selbst betriebenen Server laufen, sondern als Managed Database Service. Notiere während der Erstellung, welche Angaben du in einem echten Projekt kennen müsstest, bevor du migrierst.

Erstelle eine Liste mit mindestens sieben Migrationsinformationen: Database Engine, Version, Datenbankgröße, Downtime-Fenster, Benutzername, Passwort, Netzwerkzugriff und Security Group. Ergänze, welche Information du im Lab wirklich siehst und welche du nur als Planungsfrage notierst. Schließe mit zwei technischen Sätzen ab: Welche Arbeit übernimmt RDS beim Betrieb, und welche Aufgaben bleiben bei dir als Kunde?
