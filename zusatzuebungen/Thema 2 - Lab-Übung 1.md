# Thema 2 - Lab-Übung 1

Tag: Tag 1-2

Thema: Vorteile der AWS Cloud

Übung: Speed & Agility mit EC2 Windows

Verwendetes Go-Deploy-Lab: Lab 02

Starte in Lab 2 die Windows EC2 Instance wie im Guide beschrieben. Halte während des Launch-Vorgangs fünf Entscheidungen fest: AMI, Instance Type, Key Pair, Network Settings und Security Group. Schreibe zu jeder Entscheidung in einfachen Worten, was sie für den späteren Server bedeutet. Beispiel: Das AMI bestimmt das Betriebssystem und die Grundvorlage der Instanz.

Verbinde dich anschließend per Remote Desktop Protocol (RDP), sobald das Lab diesen Schritt vorsieht. Notiere, wie lange es ungefähr gedauert hat, bis aus einer Auswahl in der Konsole ein nutzbarer Server wurde. Formuliere danach zwei Sätze zu den Cloud-Vorteilen Speed & Agility: Was ging schnell, und welche Arbeit wäre bei einem physischen Server im Keller zusätzlich nötig gewesen?
