# Thema 16 - Lab-Übung 1

Tag: Tag 14/15

Thema: Ressourcen für Fakturierung, Budget und Kostenmanagement

Übung: Kostenkontrolle ohne Billing-Konsole

Verwendetes Go-Deploy-Lab: Lab 11

Die Lab-Umgebung stellt keine vollständige Billing-Konsole bereit. Nutze deshalb Lab 11 als praktischen Kosten-Fallback: Erstelle eine S3 Lifecycle Rule und erkläre, wie technische Konfiguration spätere Kosten beeinflussen kann. Notiere, welche Objekte nach welcher Zeit in eine günstigere Storage Class wechseln oder ablaufen sollen.

Ergänze danach eine kurze Werkzeugliste für echte AWS-Kostenarbeit. AWS Pricing Calculator nutzt man für Prognosen vor dem Aufbau. AWS Budgets nutzt man für Warnungen bei Grenzwerten. Cost Explorer nutzt man für historische Analyse und Trends. Cost & Usage Report liefert sehr detaillierte Rohdaten. Schreibe zu jedem Werkzeug genau eine typische Frage, die es beantwortet, zum Beispiel: 'Was wird diese Architektur ungefähr kosten?'
