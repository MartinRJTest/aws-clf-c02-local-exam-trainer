# Thema 2 - Lab-Übung 2

Tag: Tag 1-2

Thema: Vorteile der AWS Cloud

Übung: Pay-as-you-go und Self-Service mit EC2 Linux

Verwendetes Go-Deploy-Lab: Lab 03

Starte in Lab 3 die Linux EC2 Instance. Achte beim Launch bewusst darauf, welche Schritte du selbst in der Konsole auswählst und welche Arbeit AWS danach im Hintergrund übernimmt. Notiere mindestens vier Punkte, zum Beispiel AMI-Auswahl, Instance Type, Key Pair, Netzwerk oder Security Group. Verbinde diese Punkte mit den Begriffen on-demand self service und pay-as-you-go.

Wenn du per PuTTY verbunden bist, rufe wie im Lab die Instance Metadata ab. Notiere mindestens drei Informationen, die du dort findest, etwa Instance ID, Region oder Netzwerkdaten. Erkläre anschließend kurz, warum solche Betriebsinformationen für Administration hilfreich sind, aber nicht mit Kundendaten verwechselt werden dürfen.
