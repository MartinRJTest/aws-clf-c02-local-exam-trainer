Quick guide — AWS CLF-C02 Practice Trainer

This tool gives you original-style English exam questions with explanations and an AI study assistant at your side. Here is how to use it:

1. Unlock the AI assistant (once per browser window)

Click 🔑 at the top right of the assistant panel and paste your personal Mammouth API key (found in your Mammouth account under Settings → API). The key stays stored while this browser window is open — so you do not have to re-enter it for every question. When the window is closed it is deleted automatically for security reasons.

Via the line on the start page you can change the key at any time or specifically remove only the key (your study progress is kept). As soon as no key is stored, the tool cannot make any API calls — so no further costs can arise. If you want to be entirely sure, use "Delete all local data & API key" at the bottom.

The inexpensive model gpt-5.4-mini is preset — more than sufficient for the question explanations and gentle on your prepaid credit. Use the assistant for studying around the questions and the credit will easily last.

2. Choose your study focus

In the 🎯 Study focus box you set what you want to practice. First set "My course position" at the top to the topic your course is currently covering — you will learn in class which category that is. Then pick a mode:

Current topic onlyOnly questions for the currently selected category — ideal right after a lesson.
Everything so farRecommendedAll topics from category 1 up to your course position — good for continuous review.
Custom selectionOpens a list where you can tick individual topics (multiple possible) — perfect for drilling personal weak spots.
Full examAll 14 topics mixed — the real exam situation. Use this mix mode from the last course day onward for exam training.

Below you can always see how many questions your current pool contains.

3. Practice

Choose the number of questions and start either a normal practice exam (random selection from your focus pool) or the 🤖 AI-adaptive mode: after each answer the AI picks the didactically most useful next question from your focus pool and repeats weak spots until they stick.

For each question you select your answer and confirm it. The explanation then appears immediately — correcting your answer is deliberately not possible (strict study mode). For every question you can ask the AI assistant on the right; it knows the current question, your answer and the official explanation.

4. Results

At the end of each run you see your score, a breakdown by course topic (where you are solid, where not) and your "problem questions" — questions you got wrong repeatedly. From there you can continue practicing in a targeted way.

In AI-adaptive mode two values are distinguished: exam readiness counts only what you got right on the first attempt (that is your true knowledge level), while learning performance additionally shows what you mastered through repetition during the session. The progress bar combines both: green = right on the 1st attempt, yellow = learned this session, grey = still open.

Privacy

Your study progress is stored exclusively locally in your browser (localStorage). There is no server. Only your questions to the AI assistant go to Mammouth.ai.

Settings

API key

You manage the Mammouth API key directly via the "🔑 AI assistant …" line on the start page (activate, change, remove). As long as no key is stored, the tool cannot make any API calls — so no costs can arise.

⚠ Warning — use expressly at your own risk.
This option is a pure convenience feature. If enabled, your own browser stores the API key unencrypted in localStorage on this device. Anyone with access to this device, this browser profile or the browser storage can read the key and make API calls at your expense. Never enable this option on shared, public or unattended devices.
You enable it voluntarily and at your own responsibility. The sole responsibility for the API key, its confidentiality, and for any consequences, costs or damage from loss, theft or misuse lies exclusively with you. The course, the trainer and the provider of this tool accept no liability whatsoever. When in doubt: leave the option off and re-enter the key when needed.

Back up chat history

Exports the history of your AI assistant chats for all questions as a plain text file (.txt) — as a backup for later reading. The history also survives closing the tab; if it grows very large (from about 3 MB) the oldest chats are discarded automatically.
